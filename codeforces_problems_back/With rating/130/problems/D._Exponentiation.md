<h1 style='text-align: center;'> D. Exponentiation</h1>

<h5 style='text-align: center;'>time limit per test: 2 seconds</h5>
<h5 style='text-align: center;'>memory limit per test: 64 megabytes</h5>

You are given integers *a*, *b* and *c*. Calculate *a**b* modulo *c*.

## Input

## Input

 data contains numbers *a*, *b* and *c*, one number per line. Each number is an integer between 1 and 100, inclusive.

## Output

## Output

 *a**b* mod *c*.

## Examples

## Input


```
2  
5  
40  

```
## Output


```
32  

```
## Input


```
2  
5  
26  

```
## Output


```
6  

```
