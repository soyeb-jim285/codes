<h1 style='text-align: center;'> B. Binary notation</h1>

<h5 style='text-align: center;'>time limit per test: 2 seconds</h5>
<h5 style='text-align: center;'>memory limit per test: 256 megabytes</h5>

You are given a positive integer *n*. ## Output

 its binary notation.

## Input

The only line of input data contains an integer *n* (1 ≤ *n* ≤ 106).

## Output

## Output

 the binary notation of *n* (without any leading zeros).

## Examples

## Input


```
5  

```
## Output


```
101  

```
## Input


```
13  

```
## Output


```
1101  

```
## Note

In the first example 5 = 1 * 22 + 0 * 21 + 1 * 20.
