<h1 style='text-align: center;'> A. Pentagonal numbers</h1>

<h5 style='text-align: center;'>time limit per test: 3 seconds</h5>
<h5 style='text-align: center;'>memory limit per test: 256 megabytes</h5>

Pentagonal numbers are figurate numbers which can be calculated using the formula *p**n* = (3*n*2 - *n*) / 2 (always integer). You are given *n*; calculate *n*-th pentagonal number.

## Input

The only line of input contains an integer *n* (1 ≤ *n* ≤ 100).

## Output

## Output

 the *n*-th pentagonal number.

## Examples

## Input


```
2  

```
## Output


```
5  

```
## Input


```
5  

```
## Output


```
35  

```
