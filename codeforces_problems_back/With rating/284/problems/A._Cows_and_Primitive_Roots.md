<h1 style='text-align: center;'> A. Cows and Primitive Roots</h1>

<h5 style='text-align: center;'>time limit per test: 2 seconds</h5>
<h5 style='text-align: center;'>memory limit per test: 256 megabytes</h5>

The cows have just learned what a primitive root is! Given a prime *p*, a primitive root ![](images/f203bd9f1553a2d666452cec7d6f1f4f02a817fb.png) is an integer *x* (1 ≤ *x* < *p*) such that none of integers *x* - 1, *x*2 - 1, ..., *x**p* - 2 - 1 are divisible by *p*, but *x**p* - 1 - 1 is. 

Unfortunately, computing primitive roots can be time consuming, so the cows need your help. Given a prime *p*, help the cows find the number of primitive roots ![](images/f203bd9f1553a2d666452cec7d6f1f4f02a817fb.png).

## Input

The input contains a single line containing an integer *p* (2 ≤ *p* < 2000). It is guaranteed that *p* is a prime.

## Output

## Output

 on a single line the number of primitive roots ![](images/f203bd9f1553a2d666452cec7d6f1f4f02a817fb.png).

## Examples

## Input


```
3  

```
## Output


```
1  

```
## Input


```
5  

```
## Output


```
2  

```
## Note

The only primitive root ![](images/3722298ba062e95b18705d1253eb4e5d31e3b2d1.png) is 2.

The primitive roots ![](images/1d85c6a17ef1c42b53cf94d00bc49a7ac458fd58.png) are 2 and 3.
