<h1 style='text-align: center;'> F. The Sum of the k-th Powers</h1>

<h5 style='text-align: center;'>time limit per test: 2 seconds</h5>
<h5 style='text-align: center;'>memory limit per test: 256 megabytes</h5>

There are well-known formulas: ![](images/5391ac5117a06bc4bd7abe8e188ccda9e2478758.png), ![](images/a851dad95e7e4aa94bef8930988d272774a2601c.png), ![](images/f3889d7d5f09f3428aeddd86bfda56c7e162121a.png). Also mathematicians found similar formulas for higher degrees.

Find the value of the sum ![](images/22622446a1c46d96c309737feb64ab8fed90c030.png) modulo 109 + 7 (so you should find the remainder after dividing the answer by the value 109 + 7).

## Input

The only line contains two integers *n*, *k* (1 ≤ *n* ≤ 109, 0 ≤ *k* ≤ 106).

## Output

Print the only integer *a* — the remainder after dividing the value of the sum by the value 109 + 7.

## Examples

## Input


```
4 1  

```
## Output


```
10  

```
## Input


```
4 2  

```
## Output


```
30  

```
## Input


```
4 3  

```
## Output


```
100  

```
## Input


```
4 0  

```
## Output


```
4  

```
