<h1 style='text-align: center;'> C. Array Sum</h1>

<h5 style='text-align: center;'>time limit per test: 2 seconds</h5>
<h5 style='text-align: center;'>memory limit per test: 64 megabytes</h5>

You are given an array of integers. Calculate the sum of its elements.

## Input

The *i*-th line of the input contains an integer *a**i* (0 ≤ *a* ≤ 1000) — the *i*-th element of the array. The size of the array is between 1 and 10, inclusive. Note that the size of the array is not given explicitly!

## Output

## Output

 a single integer — the sum of the elements of the array.

## Examples

## Input


```
2  
15  
110  
3  

```
## Output


```
130  

```
## Input


```
90  
0  
21  
331  
45  

```
## Output


```
487  

```
